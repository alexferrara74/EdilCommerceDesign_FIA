- - - Introduzione - - -
L’obiettivo del progetto è quello di realizzare un’agente che sia in grado di suggerire all’utente prodotti,
mediante la selezione di prodotti potenzialmente acquistabili sulla base delle transazioni commerciali precedenti.
Tale agente cercherà di aumentare le vendite dei prodotti meno acquistati, cosi da ridimensionare le quantità dei prodotti invenduti.

- - - Risorse repository - - -
Nella nostra repository sono stati eseguiti commit in merito alle componenti del progetto:
cartella colab -> algoritmi in python di analisi del clustering e di pulizia dei dati
cartella colab -> Invendum (programma da eseguire)
cartella dataset -> cartella contenente tutte le versioni del nostro dataset
